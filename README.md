# AI Chatbot & Automation Tool  

An intelligent Python-based project that integrates **ChatGPT (OpenAI API)** and other AI models to automate tasks, generate dynamic responses, and assist users through chat or console interaction.

## Features
- ChatGPT API integration for intelligent conversations  
- Task automation (text generation, summarization, content creation)  
- Modular code structure — easy to expand for Telegram, Discord, or web  
- Configurable API key & prompt templates  
- Clean and readable output formatting  

---

## Tech Stack
- **Language:** Python 3.11+  
- **Libraries:** `openai`, `requests`, `dotenv`  
- **Concepts:** API handling, JSON parsing, modular programming, AI integration  

---

## How It Works
1. The script connects to the ChatGPT API (or Grok/Perplexity).  
2. Sends user input and retrieves an AI-generated response.  
3. Optionally performs post-processing — summarization, formatting, or automation.
